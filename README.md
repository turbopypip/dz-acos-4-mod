# ACOS Module 4 Homeworks

This repository contains C solutions for ACOS module 4 homeworks from the course wiki.

## Assignments

- `dz1`: HW 1 (C)
- `dz2`: HW 2 (System calls)
- `dz3`: HW 3 (Linking and interposition)
- `dz4`: HW 4 (Strings and regular expressions)
- `dz5`: HW 5 (Processes and pipes)

## Build

```bash
cmake -S . -B build
cmake --build build
```

Executables are created under `build/dz*`.
